# Claim boundary

Glite is a pre-likelihood triage scaffold. It is designed to make early-stage scalar-tensor modified-gravity branches easier to reject, reproduce, or promote to backend implementation under declared conditions.

It does not provide observational support, posterior constraints, Bayesian evidence, a true Hu-Sawicki f(R) model ranking, or a Solar-System viability solution. The solar row is a pre-PPN toy surrogate only. The external precheck is synthetic-width diagnostic only and not an observational likelihood.
